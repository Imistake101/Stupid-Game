Stupid Cat Game

Requirements:
- Python 3.x
- Pygame library

Installation:

1. Install Python 3.x from the official website: https://www.python.org/downloads/

2. Install Pygame using pip:
   Open your terminal or command prompt and run:
   pip install pygame

Setup:
 
1. - Download and place the folder in the 'C:' directory.

Running the Game:

1. Open a terminal or command prompt.

2. Navigate to the 'C:\stupid game' directory:
   cd C:\stupid game

3. Run the game script:
   python stupid_cat.py

or just double click the stupid_cat.py

Enjoy playing the Stupid Cat Game!



(this game is obviously satire)
